## Endpoints
> {BASE} = Base API Url - https://pastr.userr00t.com

### GET {BASE}/{id}
Opens the [`view`](src/views/view.vue) page and fetches one paste by ID.
 
### GET {BASE}/
Opens the [`create/edit`](src/views/create.vue) page.
